# 💬 Morse Code Translator 🔁

A stylish and responsive Morse Code Translator built with HTML, CSS, and JavaScript.

Translate between:
- **Text ➡️ Morse**
- **Morse ➡️ Text**

All in one clean, responsive web interface! 🚀

---

## ✨ Features

✅ Convert text to Morse code  
✅ Convert Morse code back to text  
✅ Responsive design (mobile-friendly)  
✅ Stylish interface with glowing effects  
✅ Easy to use, no setup required

---

## 🛠️ Technologies

- HTML5
- CSS3 (custom styles + Google Fonts)
- JavaScript (vanilla)
